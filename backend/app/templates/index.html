Лаконичный рандомайзер

Наглядный и простой. Может генерировать серии чисел без повторов, не теряет данные при перезагрузке страницы.

Выбор победителя из списка

Работает на больших списках (проверено до 1 000 000 строк). Есть фильтр дубликатов. Результаты не храним.

Рандомусspecial

Инструмент для проведения розыгрышей призов онлайн, в один или несколько этапов, с возможностью настройки механики и внешнего вида.

Разделить список на группы

Делит список на заданное количество групп или на группы с заданным количеством участников.

Конкурс ВКонтакте

Считаем лайки, репосты и комментарии к записям, проверяем наличие в подписчиках/друзьях в любых комбинациях. Храним все результаты.

Случайные числа с картинкой и ссылкой на результат

Результат сохраним. У него будет постоянная ссылка и готовая картинка.

Розыгрыш с описанием

Честный розыгрыш c помощью генератора случайных чисел. Подвести итоги можно только один раз. Подменить результаты невозможно. Данные хранятся бессрочно.

Генератор имен и фамилий

Для русских и английских имен в различных форматах, с учетом реальной частоты использования в современном мире.

Генератор случайных ников

Генератор паролей

Генератор промокодов для акций и распродаж

Настриваемый формат промокода и списка для выгрузки результатов.

Поиск слов из 5 букв.

Генератор гороскопов

Выбор случайного ответа

Индивидуальные проекты
